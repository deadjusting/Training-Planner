## Описание программы

*"Training Planner — это GUI-приложение для планирования тренировок. Позволяет добавлять записи о тренировках, фильтровать их по типу и дате, а также сохранять данные в формат JSON."*
---

## Требования для запуска

Python 3.10 или выше
Библиотеки: tkinter (входит в стандартную библиотеку Python)

---

## Как запустить

```bash
git clone https://github.com/deadjusting/training-planner.git
cd training-planner
python main.py

## Как пользоваться
1.	При запуске открывается окно приложения с полями для ввода:
2.	Заполните поля и нажмите кнопку «Добавить тренировку», чтобы добавить запись в таблицу с тренировками.
4.	Используйте фильтры для поиска тренировок по типу или дате.
5.	Все данные автоматически сохраняются в файле data.json.

## Пример работы
Добавление тренировки:
Введите дату (ГГГГ-ММ-ДД): 2026-04-20
Введите тип тренировки: Бег
Введите длительность (в минутах): 30
✅ Тренировка добавлена!

Фильтрация по типу тренировки:
Тип тренировки: Бег
2026-04-20 | Бег | 30 минут

Тип теста	Пример				Результат
