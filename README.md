## Описание программы

*Training Planner* — это GUI-приложение для планирования тренировок. Позволяет добавлять записи о тренировках, фильтровать их по типу и дате, а также сохранять данные в формате JSON для последующего использования и редактирования.

---

## Требования для запуска

- Python 3.10 или выше
- Библиотека tkinter (входит в стандартную поставку Python)

---

## Как запустить

```bash
git clone https://github.com/твой-логин/training-planner.git
cd training-planner
python main.py
